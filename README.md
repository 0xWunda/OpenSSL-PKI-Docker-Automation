# OpenSSL-PKI-Docker-Automation
OpenSSL-PKI-Docker-Automation ist ein komplett automatisiertes Toolkit zur Erstellung einer vollständigen Public Key Infrastructure (PKI) mit Root-CA, Intermediate-CA, Server- und Client-Zertifikaten, inklusive CRL-Management, OCSP-Responder und einer vollständig integrierten Nginx-Docker-Umgebung für mTLS-basiertes Testing.
